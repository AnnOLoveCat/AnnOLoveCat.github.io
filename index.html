Discord 新聞發布機器人（Python）

我用 discord.py 做了一個新聞推送 Bot：定時抓取新聞、去重、整理成 Embed， 並支援 Slash 指令手動抓取與設定推送頻道。
目前穩定來源：LTN（自由時報 RSS + 文章內頁解析）；其他來源先保留為後續擴充。

已完成功能（目前可用）

  • 每小時自動推送：tasks.loop(hours=1)
  • 去重避免洗版：用 set 記錄 title-url
  • Embed 美化：標題 + 連結 + timestamp + 縮圖
  • 可設定頻道：/set_news_channel
  • 手動抓新聞：/fetch_latest_news（目前以 LTN 為主）

技術棧

Python discord.py app_commands tasks.loop feedparser requests BeautifulSoup dotenv

來源以 NEWS_SOURCES 模組化管理；頻道取得採用「快取優先 → fetch_channel fallback」， 提升穩定性。

LTN 抓取流程

  1. 1 讀取 LTN RSS：https://news.ltn.com.tw/rss/all.xml
  2. 2 逐篇進入文章頁：抓報導封面圖 + 文章標題
  3. 3 若無圖片則跳過（避免 Embed 空內容）
  4. 4 標題裁切（shorten）＋ 產出標準化 dict
  5. 5 由舊到新發送（reversed）＋ set 去重

Featured Project

AI Image Classifying（Streamlit × MobileNetV2）

我用 Streamlit 做了一個「影像分類」網頁 Demo：使用者上傳圖片後， 透過 Keras MobileNetV2（ImageNet 預訓練）推論，輸出 Top-3 類別與信心分數。
特色包含：完整影像前處理流程（RGB/Resize/Normalize）與 模型快取（避免每次重整重新載入模型）。

已完成功能

  • 圖片上傳：支援 JPG / PNG
  • 即時預覽：上傳後立即顯示圖片
  • 一鍵推論：按下按鈕才開始跑模型（避免浪費資源）
  • Top-3 結果：顯示前三名分類 + 機率
  • 模型快取：st.cache_resource 提升重整速度

技術棧

Python Streamlit Keras MobileNetV2 OpenCV NumPy PIL

前處理包含：RGB 轉換、224×224 Resize、preprocess_input 正規化、expand_dims 增加 batch 維度；推論後用 decode_predictions(top=3) 轉成可讀標籤。

推論流程（對應程式邏輯）

  1. 1 載入 MobileNetV2（ImageNet weights）
  2. 2 讀取上傳圖片 → 強制轉成 RGB
  3. 3 Resize 到 224×224（模型輸入規格）
  4. 4 preprocess_input 正規化 + expand_dims
  5. 5 model.predict → decode_predictions 取 Top-3
  6. 6 顯示 Label + Score（百分比）
