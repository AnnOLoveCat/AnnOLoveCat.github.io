Discord 新聞發布機器人（Python）

我用 discord.py 做了一個新聞推送 Bot：定時抓取新聞、去重、整理成 Embed， 並支援 Slash 指令手動抓取與設定推送頻道。
目前穩定來源：LTN（自由時報 RSS + 文章內頁解析）；其他來源先保留為後續擴充。

已完成功能（目前可用）

  • 每小時自動推送：tasks.loop(hours=1)
  • 去重避免洗版：用 set 記錄 title-url
  • Embed 美化：標題 + 連結 + timestamp + 縮圖
  • 可設定頻道：/set_news_channel
  • 手動抓新聞：/fetch_latest_news（目前以 LTN 為主）

技術棧

Python discord.py app_commands tasks.loop feedparser requests BeautifulSoup dotenv

來源以 NEWS_SOURCES 模組化管理；頻道取得採用「快取優先 → fetch_channel fallback」， 提升穩定性。

LTN 抓取流程

  1. 1 讀取 LTN RSS：https://news.ltn.com.tw/rss/all.xml
  2. 2 逐篇進入文章頁：抓報導封面圖 + 文章標題
  3. 3 若無圖片則跳過（避免 Embed 空內容）
  4. 4 標題裁切（shorten）＋ 產出標準化 dict
  5. 5 由舊到新發送（reversed）＋ set 去重
